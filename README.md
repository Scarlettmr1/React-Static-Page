How to Run the App
Step 1: Install Dependencies
bashcd tortoise-hare-story
npm install
Step 2: Start the Development Server
bashnpm start
Component Details:

Header.js

Displays the title: "The Tortoise and the Hare"
Includes a subtitle: "A Classic Aesop's Fable"
Styled with green background

Content.js

Contains 3 paragraphs summarizing the story:

The Hare's boasting and the challenge
The race and the Hare's nap
The Tortoise's victory and lesson learned

Footer.js

Shows the moral: "Slow and steady wins the race"
Includes an explanation of the lesson
Styled with blue background
